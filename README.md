# AASTEEN — Web

Real Next.js 15 scaffold: config, layout, SEO metadata, fonts, and a working
hero section with the interactive 3D metallic logo (React Three Fiber).

## Run it
npm install
npm run dev

## What's real vs next
Built and working: project config, layout/fonts/SEO, Hero section + 3D logo.
Not yet built: shop/product/cart/checkout/account pages, Razorpay integration,
admin dashboard, order emails/database. These need a real backend and your
Razorpay keys before they can be "production" rather than code.
